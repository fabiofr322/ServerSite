Como Entrar

Entre no servidor pelo Minecraft Java ou Bedrock (Celular, Tablet e Consoles).

1

Abra o seu Minecraft

Use Java (versão 1.21.1+) ou Bedrock (última versão). O suporte crossplay permite jogar juntos!

2

Adicione o Servidor

Vá em Multiplayer (Java) ou Servidores (Bedrock) e clique em Adicionar Servidor.

3

Copie os dados de conexão

Use os botões abaixo para copiar o IP e a porta de acordo com a sua versão.
